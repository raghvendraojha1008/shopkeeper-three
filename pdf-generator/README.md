# shopkeeper-pdf-generator

Native PDF generation for Android using pdfDocument

## Install

To use npm

```bash
npm install shopkeeper-pdf-generator
````

To use yarn

```bash
yarn add shopkeeper-pdf-generator
```

Sync native files

```bash
npx cap sync
```
